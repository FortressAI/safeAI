Test AI agents against Abstract Reasoning Corpus challenges

How to Use This Demo

The ARC Prize Challenge demo allows you to test AI agents against the Abstract Reasoning Corpus, a collection of visual reasoning tasks. Follow these steps to get started:

1. Set Up Training Data

Enter the URL for your training data directory. This should point to a collection of ARC training puzzles in JSON format.

Example: https://arc-data.domain.com/training/
Tip: Training data typically includes puzzle pairs with known solutions to help the AI learn patterns.

2. Configure Evaluation Data

Enter the URL for your evaluation data directory. This contains the test puzzles that will assess your AI's performance.

Example: https://arc-data.domain.com/evaluation/
Tip: Evaluation puzzles are separate from training puzzles and test the AI's ability to generalize.

3. Start Processing

Click the "Start Processing" button to begin. The demo will:

  • First process the training phase (50 puzzles)
  • Automatically transition to the evaluation phase
  • Display results in real-time as puzzles are solved

4. Interpret Results

Each result card shows:

  • Input Grid: The initial puzzle state
  • Expected Output: The correct solution
  • Prediction: Your AI's attempted solution
  • Score: Accuracy of the prediction (0-100)
  • Agents: AI components used in solving
